# Jan's Tree Game

A bilingual (English / 中文) game for learning to identify the most common trees of the UK — spaced-repetition practice, three play modes, a browsable species library, and a quiet Wagner mode hidden in the settings.

**Play it:** open the published page, then tap **Add to home screen** to install it like an app.

A single self-contained HTML file — all artwork and audio generated in-page; nothing is loaded from a server.
